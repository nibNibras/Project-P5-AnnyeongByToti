# 📦 AnnyoengByToti: Jastip Web Platform

A sleek, responsive e-commerce web application designed to simplify international personal shopping (Jasa Titip). Built entirely with native web technologies, users can browse overseas products, learn about our safe shipping process, and connect with trusted personal shoppers.

# 🌐 Website Sections
The platform is cleanly structured into four main interactive sections:

- <strong>🏠 Home:</strong> Features a vibrant, modern hero section with our main slogan ("Titip Barang? Gampang Banget Di Sini!"), a quick introduction, and a custom interactive "Titip Sekarang" call-to-action button.

- <strong>✨ About Us:</strong> Highlights our business mission (Why Choose Us?) and explains how we make overseas shopping completely hassle-free.

- <strong>🛍️ Products:</strong> A grid display highlighting the latest highly-requested trending items, complete with favorite (heart) toggles and shopping cart logic.

- <strong>📞 Contact Us:</strong> A clean, functional section allowing users or potential shoppers to reach out to the support team.
